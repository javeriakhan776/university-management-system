# University Management System (Phase 1)

A simple **Object-Oriented Programming (OOP)** project built in Java as my first structured multi-file project using an IDE (IntelliJ IDEA).

This project models a basic university system with core entities such as **Student** and **Course**, focusing on applying fundamental OOP principles in a practical way.

---

## 📌 Project Overview

This system allows basic interaction between students and courses, including:

- Creating student and course objects
- Enrolling students into courses
- Managing relationships between entities
- Displaying relevant information about students and courses

The goal of this phase was not to build a complete system, but to **understand and apply OOP concepts in a real project structure**.

---

## 🧠 Key OOP Concepts Applied

- Encapsulation (private fields with controlled access)
- Class relationships (Student ↔ Course interaction)
- Constructor-based initialization
- Method-based behavior modeling
- Basic data protection and structure design
